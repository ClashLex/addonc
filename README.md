# addonc

A collection of responsive HTML card components with light and dark theme support.

## Features

- 📱 Responsive design
- 🌙 Dark mode support
- ⚡ Lightweight and fast
- 🎨 Easy to customize

## Files

- `card.html` - Light theme card component
- `card2dark.html` - Dark theme card component
- `card2dark.js` - Dark theme toggle functionality

## Usage

Simply copy the HTML/CSS code from the respective files and paste into your project.
